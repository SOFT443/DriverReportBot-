# Driver Bot

Telegram бот для таксопарка Драйвер (Москва).

## Функционал
- Водитель вводит госномер ТС
- Бот проверяет номер по списку
- Водитель вводит график работы
- Обращение отправляется двум менеджерам

## Команды
- /start - начать диалог
- /cancel - отменить

## Деплой на Render
1. Создать новый репозиторий на GitHub
2. Загрузить все файлы
3. В Render создать Web Service, подключить репозиторий
4. Build Command: `pip install -r requirements.txt`
5. Start Command: `python main.py`
